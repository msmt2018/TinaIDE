# 常见问题

## 开源版还能使用哪些在线能力

反馈、插件浏览、插件安装、包下载、收藏、下载历史和 GitHub Release 版本更新提示仍然保留。

## 为什么没有私有身份和商业功能

这些能力已经从公开仓库移除。私有后端和管理端不随开源项目分发。

## AI 怎么使用

在 AI 设置中使用 Custom BYOK，配置你自己的服务商地址、模型和 API Key。

## 插件市场为什么只保留公开仓库能力

开源版市场读取 GitHub Registry。客户端负责浏览、安装和更新插件，不包含私有账号互动和审核流程。
