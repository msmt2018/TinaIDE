# 开源版我的页面

开源版 TinaIDE 已迁出私有身份和商业功能。

这个页面现在聚焦本地入口和公开能力：

- 反馈与问题上报
- 设置中心
- 关于与日志
- 下载历史
- 插件、包管理和本地数据入口

## 可以做什么

你可以从这里进入反馈页提交问题，也可以查看版本信息、日志工具和下载历史。

插件浏览、插件下载、包下载、收藏和下载历史仍然保留。需要服务端身份的发布、评分、评论和举报能力在开源版中不可用。
